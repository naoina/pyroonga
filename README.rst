pyroonga
========

What's this?
------------
Python interface for `groonga`_ fulltext search engine.

Requirements
------------

- Python 2.6 or 3.x and later
- groonga

Installation
------------

from pypi::

   % pip install pyroonga

from source::

   % python setup.py install
